# Airbnb NYC Data Cleaning

## Project Description
This project focuses on cleaning the Airbnb New York City dataset using Python. It includes handling missing values, removing duplicates, correcting data types, and preparing the dataset for further analysis.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Dataset
AB_NYC.csv

## Project Features
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Data Type Conversion
- Data Preparation
