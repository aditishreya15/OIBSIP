# Wine Quality Prediction

## Project Description
This project predicts the quality of wine using machine learning classification algorithms. It analyzes various chemical properties of wine, such as acidity, density, pH, and alcohol content, to classify wine quality accurately.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Random Forest Classifier
- Stochastic Gradient Descent (SGD) Classifier
- Support Vector Classifier (SVC)

## Dataset
Wine Quality Dataset

## Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training
- Model Evaluation
- Wine Quality Prediction
