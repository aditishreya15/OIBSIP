# Retail Sales Analysis

## Project Description
This project analyzes retail sales data using Python. It includes data cleaning, exploratory data analysis (EDA), and visualization to gain business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Retail Sales Dataset

## Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Product Category Analysis
- Data Visualization
