# Sentiment Analysis

## Project Description
This project performs sentiment analysis on text data using Natural Language Processing (NLP) techniques. The goal is to classify text into positive, negative, or neutral sentiments through data preprocessing, feature extraction, model training, and evaluation.

## Tools Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Sentiment Analysis Dataset

## Project Features
- Data Preprocessing
- Text Cleaning
- Tokenization
- Stopword Removal
- Sentiment Classification
- Model Evaluation
- Data Visualization
