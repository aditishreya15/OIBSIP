# Predicting House Prices with Linear Regression

## Project Description
This project predicts house prices using the Linear Regression algorithm. It involves data preprocessing, exploratory data analysis (EDA), feature selection, model training, prediction, and evaluation to estimate house prices based on different property features.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
House Price Dataset

## Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Linear Regression Model
- Model Evaluation
- House Price Prediction
